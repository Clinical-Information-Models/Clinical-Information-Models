Value sets have been generated using either the time trade-off (TTO) valuation technique or the visual analogue scale (VAS) valuation technique. A number of countries/regions have both TTO and VAS value sets for EQ-5D-3L. A list of all currently available value sets for the EQ-5D-3L can be found in the table below. For instructions on how to select the most appropriate value set for your study, please see Choosing a value set .
